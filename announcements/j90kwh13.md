# 馬祖到底有什麼特色？

如果只能用一句話形容：

> 馬祖 = 戰地遺跡 + 閩東文化 + 海蝕地景 + 藍眼淚

它其實不像澎湖（海灘）
也不像小琉球（浮潛）

而是一種：

**「被遺忘的海上軍事王國」**

這才是馬祖最獨特的地方。
